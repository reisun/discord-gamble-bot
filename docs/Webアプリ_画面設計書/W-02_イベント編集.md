# W-02: イベント編集（作成・編集兼用）

**ハッシュパス**: `#/dashboard/:guildId/new-event` / `#/dashboard/:guildId/:eventId/edit`  
**パンくず（新規）**: ホーム > 新規作成  
**パンくず（編集）**: ホーム > [イベント名] > 編集

## 画面構成

タイトル（h1）はカード内の最上部に表示する。パンくずの直下にはカードのみが続くレイアウト。

```
┌──────────────────────────────────────────────────────┐
│  新規イベント作成 / イベント編集  ← カード内タイトル  │
│                                                      │
│  イベント名      [________________________]          │
│  初期ポイント    [__________] pt                     │
│                                                      │
│              [ キャンセル ]  [ 保存 ]                │
└──────────────────────────────────────────────────────┘
```

## フォーム項目

| フィールド | 種類 | 必須 | バリデーション |
|------------|------|------|---------------|
| イベント名 | テキスト | ○ | 1〜100文字 |
| 初期ポイント | 数値 | ○ | 1以上の整数（デフォルト: 10000） |
