# Discord Bot 環境変数

| 変数名 | 必須 | 説明 |
|--------|------|------|
| `DISCORD_BOT_TOKEN` | ○ | Discord Bot のトークン |
| `DISCORD_CLIENT_ID` | ○ | Discord アプリケーション ID |
| `DISCORD_GUILD_ID` | ○ | 対象の Discord サーバー ID |
| `DISCORD_ADMIN_ROLE_ID` | ○ | 管理者ロールの ID（複数の場合はカンマ区切り） |
| `API_BASE_URL` | ○ | WebサーバーのベースURL（内部通信用。例: `http://server:3000`） |
| `WEB_APP_BASE_URL` | ○ | WebアプリのベースURL（`/dashboard` コマンドのボタンリンク生成用） |
| `ADMIN_TOKEN` | ○ | 管理者用トークン（サーバーAPI認証用。推測困難なランダム文字列を設定すること） |
