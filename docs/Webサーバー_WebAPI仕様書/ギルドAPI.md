# ギルド API

## ギルド情報取得
```
GET /api/guilds/:guildId
```

指定した Discord サーバーのギルド情報を返す。

**パスパラメータ**

| パラメータ | 説明 |
|-----------|------|
| guildId | DiscordサーバーID |

**レスポンス**
```json
{
  "data": {
    "guildId": "123456789012345678",
    "guildName": "テストサーバー"
  }
}
```

**エラーレスポンス**

| HTTPステータス | コード | 説明 |
|--------------|--------|------|
| 404 | `NOT_FOUND` | 指定IDのギルドが存在しない |

---

## ギルド情報登録・更新
```
PUT /api/guilds/:guildId
```
**権限**: 管理者のみ（Discord Bot が呼び出す）

Discord Bot の起動時に各サーバーのギルド情報を登録または更新する。既存レコードがある場合は `guild_name` を更新する（UPSERT）。

**パスパラメータ**

| パラメータ | 説明 |
|-----------|------|
| guildId | DiscordサーバーID |

**リクエストボディ**
```json
{
  "guildName": "テストサーバー"
}
```

| フィールド | 型 | 必須 | 説明 |
|-----------|-----|------|------|
| guildName | string | ○ | Discordサーバー名 |

**レスポンス**: `200 OK`
```json
{
  "data": {
    "guildId": "123456789012345678",
    "guildName": "テストサーバー"
  }
}
```
