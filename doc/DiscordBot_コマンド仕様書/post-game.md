# `/post-game` - ゲーム情報をチャンネルに投稿する

## 説明
指定したゲームの情報（タイトル・賭け項目・締め切り）を現在のチャンネルへ全員向けに投稿する。

## オプション

| オプション名 | 型 | 必須 | 説明 |
|------------|-----|------|------|
| `game` | Integer | ○ | ゲーム番号（オートコンプリートで公開中ゲームが候補表示される） |

## 処理フロー

1. 管理者ロールを確認（なければ拒否）
2. 開催中イベントのゲーム一覧を取得し、ゲーム番号（1始まりの並び順）でゲームを特定する
3. ゲームが公開中か確認
4. Discord Embed に整形してチャンネルへ送信（全員表示）

## 投稿メッセージ（全員表示 / Embed）

Discord の Embed 形式で投稿する。色: Discord Blurple (#5865F2)

| Embed フィールド | 内容 | 条件 |
|----------------|------|------|
| タイトル | `🎮 #<ゲーム番号> <ゲームタイトル>` | 常時 |
| 説明文 | ゲームの description | description が null でない場合のみ |
| 賭け方式 (inline) | `複数-順番一致（重複なし）` など | single 以外のみ |
| 締め切り (inline) | `2024-01-01 21:00` | 常時 |
| 賭け項目 | `` `A` チームA `` を改行区切りで列挙 | 常時 |
| 賭け方法 | `/bet` コマンドの使い方と例 | 常時 |

**単数方式の場合（例）**

Embed タイトル: `🎮 #1 第1試合`  
Embed 説明文: `チームAとチームBの対戦です。`

| フィールド名 | 値 |
|------------|---|
| 締め切り | 2024-01-01 21:00 |
| 賭け項目 | `` `A` チームA `` <br> `` `B` チームB `` |
| 賭け方法 | `` `/bet game:1 option:<記号> amount:<ポイント>` `` <br> `例: /bet game:1 option:A amount:100` |

**複数方式の場合（例: 複数-順番一致 / 3択）**

Embed タイトル: `🎮 #1 第1試合`

| フィールド名 | 値 |
|------------|---|
| 賭け方式 | 複数-順番一致（重複なし） / 選択数: 3 |
| 締め切り | 2024-01-01 21:00 |
| 賭け項目 | `` `A` チームA `` <br> `` `B` チームB `` <br> ... |
| 賭け方法 | `` `/bet game:1 option:<記号を3文字> amount:<ポイント>` `` <br> `例: /bet game:1 option:ABC amount:100` |

## エラー時（Ephemeral）

| 条件 | メッセージ |
|------|-----------|
| ゲームが存在しない | 指定されたゲームが見つかりません。 |
| ゲームが非公開 | 非公開のゲームは投稿できません。 |
| 管理者権限なし | このコマンドは管理者のみ使用できます。 |
