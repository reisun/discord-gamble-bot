# W-01: イベント一覧

**ハッシュパス**: `#/events`  
**パンくず**: ホーム > イベント一覧

## 画面構成
```
[ + 新規イベント作成 ]  (管理者のみ表示)

┌────────────────────────────────────────────────────────────────────────────┐
│  イベント名          開催状態     公開      操作 (管理者のみ)              │
├────────────────────────────────────────────────────────────────────────────┤
│  〇〇大会            開催中 ●    公開      [ゲーム一覧] [編集] [削除] [開催中切替] [非公開にする] │
│  △△カップ           ー           非公開    [ゲーム一覧] [編集] [削除] [開催中切替] [公開にする]  │
│  ...                                                                        │
└────────────────────────────────────────────────────────────────────────────┘
```

※「公開」列は管理者のみ表示

## 要素詳細

| 要素 | 表示条件 | 動作 |
|------|----------|------|
| `+ 新規イベント作成` ボタン | 管理者のみ | W-02（新規）へ遷移 |
| `ゲーム一覧` ボタン | 全員 | W-03 へ遷移 |
| `編集` ボタン | 管理者のみ | W-02（編集）へ遷移 |
| `削除` ボタン | 管理者のみ | 確認ダイアログ後削除 |
| `開催中切替` ボタン | 管理者のみ | 開催状態をトグル。非開催→開催時は `is_published` を強制公開。開催0件も許容 |
| `公開にする` / `非公開にする` ボタン | 管理者のみ | 公開状態をトグル。開催中は `disabled`（非公開不可） |
| 開催状態バッジ | 全員 | 開催中は `●開催中`、非開催は `ー` を表示 |
| 公開列 | 管理者のみ | `公開` / `非公開` を表示 |

## 一覧の表示フィルタ

| ユーザー種別 | 表示対象 |
|-------------|---------|
| 管理者 | 全イベント（公開・非公開問わず） |
| 一般ユーザー | `is_published = TRUE` のイベントのみ |
