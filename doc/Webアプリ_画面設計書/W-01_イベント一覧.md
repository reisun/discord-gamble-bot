# W-01: イベント一覧

**ハッシュパス**: `#/events`  
**パンくず**: ホーム > イベント一覧

## 画面構成
```
[ + 新規イベント作成 ]  (管理者のみ表示)

┌──────────────────────────────────────────────────────┐
│  イベント名          開催状態     操作               │
├──────────────────────────────────────────────────────┤
│  〇〇大会            開催中 ●    [ゲーム一覧] [編集] [削除] [開催中切替] │
│  △△カップ           終了         [ゲーム一覧] [編集] [削除] [開催中切替] │
│  ...                                                  │
└──────────────────────────────────────────────────────┘
```

## 要素詳細

| 要素 | 表示条件 | 動作 |
|------|----------|------|
| `+ 新規イベント作成` ボタン | 管理者のみ | W-02（新規）へ遷移 |
| `ゲーム一覧` ボタン | 全員 | W-03 へ遷移 |
| `編集` ボタン | 管理者のみ | W-02（編集）へ遷移 |
| `削除` ボタン | 管理者のみ | 確認ダイアログ後削除 |
| `開催中切替` ボタン | 管理者のみ | 開催中イベントを切り替え（1件のみ開催中） |
| 開催状態バッジ | 全員 | 開催中は `●` などで視覚的に区別 |
