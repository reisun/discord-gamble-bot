# W-05: ゲーム画面

**ハッシュパス**: `#/dashboard/:guildId/:eventId/:gameId`  
**パンくず**: ホーム > [イベント名] > [ゲームタイトル]

## 画面構成

**管理者向け表示（例: 複数-順番一致 3連）**
```

このゲームの説明文が入ります（設定されている場合のみ表示）
締め切り: 2024-01-01 12:00  (残り XX分)
状態: 受付中 / 締め切り済 / 結果確定

── 賭け項目 ────────────────────────────────────────────────
  A: チームA  B: チームB  C: チームC  D: チームD  E: チームE

── 賭け組み合わせ・倍率 ─────────
┌──────────────────────────────────────────────────────────┐
│  BDE: チームB→チームD→チームE       ×1.67倍          │
│                                      600pt / 3人       │

│                                      400pt / 2人       │
└──────────────────────────────────────────────────────────┘

── ゲーム結果 ──────────────────────────────────────────────
  確定済みの場合: 「当選: BDE（チームB→チームD→チームE）」と表示
  未確定の場合: 「結果未確定」と表示

結果を確定する（締め切り後に有効化）:
  [単数]       当選項目を選択: [ A: チームA ▼ ]
  [複数-選択一致]  当選項目を選択（順不同）:

                   ※ 選択数: 3 に達したら「確定」ボタンが有効化
  [複数-順番一致] 1位: [ A ▼ ]  2位: [ B ▼ ]  3位: [ D ▼ ]  （選択数分のドロップダウン）

[ 確定 ]
```

## 組み合わせ表示フォーマット

| 方式 | 組み合わせ表示例 |
|------|----------------|
| `single` | `A: チームA` |
| `multi_unordered` | `BDE: チームB、チームD、チームE` |
| `multi_ordered` / `multi_ordered_dup` | `BDE: チームB→チームD→チームE` |

## 要素詳細

| 要素 | 表示条件 | 説明 |
|------|----------|------|
| `編集` ボタン | 管理者のみ | ゲームタイトル横。W-04（編集）へ遷移 |
| `削除` ボタン | 管理者のみ | ゲームタイトル横。未公開ゲーム、または締め切り後の公開済みゲームでのみ押下可能。確認ダイアログ後削除し W-03 へ遷移 |
| `公開する` / `非公開にする` ボタン | 管理者のみ | ゲームタイトル横。公開状態をトグル |
| `即時締め切り` ボタン | 管理者のみ（公開中かつ受付中のとき） | ゲームタイトル横。締め切り日時を現在時刻+1分に更新する |
| 賭け方式・選択数バッジ | 全員 | ゲームタイトル横に表示 |
| ゲーム説明 | 全員（設定時のみ） | タイトル直下に表示。`description` が `null` の場合は非表示 |
| 賭け項目リスト | 全員 | ゲーム情報エリア内に表示。`symbol: label` 形式でタグ状に並べる |
| 残り時間 | 締め切り前 | カウントダウン表示 |
| 倍率 | 全員 | 組み合わせ単位のパリミュチュエル方式で算出 |
| 賭け総数・参加人数 | 管理者: 常時 / 一般: 結果確定後のみ | - |
| 結果確定フォーム | 管理者のみ | 締め切り後に有効化。方式に応じたUI（ドロップダウン/チェックボックス） |
| 結果列 | 全員 | 結果確定後のみ表示 |
| 個人賭け一覧の名前 | 管理者のみ | Discordサーバーニックネームを表示。未設定の場合はグローバル表示名にフォールバック |
| 個人賭け一覧のアバター | 管理者のみ | 名前の左にDiscordアイコン（20px丸形）を表示。未設定の場合はグレーのプレースホルダー |

## 倍率計算式（パリミュチュエル方式）
```
組み合わせの倍率 = 総賭けポイント合計 ÷ その組み合わせへの賭けポイント合計
当選時獲得ポイント = 賭けたポイント × 倍率（小数点以下切り捨て）
```
賭けが集まるほど倍率が下がる。倍率は賭けが入るたびにリアルタイムで変動する。
