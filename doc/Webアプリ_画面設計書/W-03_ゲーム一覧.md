# W-03: ゲーム一覧

**ハッシュパス**: `#/events/:eventId/games`  
**パンくず**: ホーム > イベント一覧 > [イベント名] > ゲーム一覧

## 画面構成
管理者表示:
```
イベント: 〇〇大会（開催中）

[ ユーザー結果一覧 ]  (管理者のみ表示)      [ + 新規ゲーム作成 ]  (管理者のみ表示)

┌──────────────────────────────────────────────────────────────────┐
│  # │ ゲームタイトル  │ 締め切り        │ 公開  │ 操作             │
├──────────────────────────────────────────────────────────────────┤
│  1 │ 第1試合          │ 2024-01-01 12:00│ 公開  │ [状況] [編集] [削除] [公開切替] │
│  2 │ 第2試合          │ 2024-01-01 13:00│ 非公開│ [状況] [編集] [削除] [公開切替] │
│ ...│                  │                 │       │                  │
└──────────────────────────────────────────────────────────────────┘
```

一般ユーザー表示（「公開」列は非表示）:
```
イベント: 〇〇大会（開催中）

┌────────────────────────────────────────────┐
│  # │ ゲームタイトル  │ 締め切り        │ 操作  │
├────────────────────────────────────────────┤
│  1 │ 第1試合          │ 2024-01-01 12:00│ [状況] │
│  2 │ 第2試合          │ 2024-01-01 13:00│ [状況] │
│ ...│                  │                 │       │
└────────────────────────────────────────────┘
```

## 要素詳細

| 要素 | 表示条件 | 動作 |
|------|----------|------|
| `ユーザー結果一覧` ボタン | 管理者のみ | W-06 へ遷移 |
| `+ 新規ゲーム作成` ボタン | 管理者のみ | W-04（新規）へ遷移 |
| `公開` 列（ヘッダー・値） | 管理者のみ | - |
| `状況` ボタン | 全員 | W-05 へ遷移 |
| `編集` ボタン | 管理者のみ | W-04（編集）へ遷移 |
| `削除` ボタン | 管理者のみ | 確認ダイアログ後削除 |
| `公開切替` ボタン | 管理者のみ | 公開・非公開を切り替え |
