# プライバシーポリシー

本サービスは、[Discord Developer Terms of Service](https://discord.com/developers/docs/policies-and-agreements/developer-terms-of-service) および [Discord Developer Policy](https://discord.com/developers/docs/policies-and-agreements/developer-policy) に基づき、以下のように定めます。

## 1. 取得する情報

本サービスは、Discord との連携を通じて以下の情報を取得します。

### ユーザー情報

| 情報 | 取得タイミング |
|------|--------------|
| Discord ユーザーID | Bot /bet コマンド使用時 |
| 表示名 | 同上 |
| アバター画像URL | 同上 |

### 認証情報

Discord OAuth2 によるログイン時に、Discord ユーザーID と表示名がセッション情報として一時的に保存されます。セッションは48時間で期限切れとなり、期限切れ後に自動削除されます。

### サーバー情報

サーバーID、サーバー名、サーバーアイコンは Bot がサーバーに参加している間保持されます。

## 2. 用途

取得した情報は、以下の目的にのみ使用します。

- ゲーム画面やランキングでのユーザー表示（表示名・アバター）
- Discord OAuth2 によるログインとアクセス制御（ユーザーID）
- サーバーごとのイベント管理とダッシュボード表示（サーバー情報）

取得した情報を第三者に提供・販売することはありません。

## 3. 禁止用途

取得した情報について、以下の行為は一切行いません。

- AI・機械学習モデルのトレーニングへの使用
- 広告ネットワークやデータブローカーへの提供
- ユーザー情報の販売・ライセンス供与・商業利用
- 匿名化されたデータの再識別

## 4. 保存期間と削除

| 対象 | 保存期間 | 削除方法 |
|------|---------|---------|
| ユーザー情報 | 登録から2週間 | Discord ID は匿名化、表示名・アバターURL は削除 |
| 認証情報 | 48時間 | 期限切れ後にレコードごと削除 |
| サーバー情報 | Bot がサーバーに参加している間 | - |

削除処理はサーバー起動時および24時間ごとに自動実行されます。

## 5. データ削除リクエスト

自動削除を待たずにユーザー情報の即時削除を希望する場合は、以下の GitHub Issues から削除リクエストを送信してください。

**[データ削除リクエスト](https://github.com/reisun/discord_gamble_bot/issues)**

リクエストには Discord ユーザーIDまたは表示名を記載してください。開発者が確認の上、速やかに対象データを削除します。
